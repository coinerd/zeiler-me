Die neue Version des Webs (2.0) bedeutet nicht, dass hier eine völlig neue Technologie zum Einsatz kommt. Denn erstens ist es nicht eine Technologie - es sind mehrere - und zweitens gibt es diese Technologien bereits seit einer ganzen Weile und drittens ist das Web 2.0 auch eine Art neue Sichtweise auf das Internet.

## Die Definition

Tim O'Reilly hat den Begriff "Web 2.0" geprägt und beschreibt damit eine zweite Generation des World Wide Webs, die sich durch die Nutzung des Internets als Plattform auszeichnet. Anstatt Software lokal auf dem eigenen Rechner zu installieren, werden Anwendungen direkt im Browser ausgeführt.

## Die Technologien

Die zentralen Technologien des Web 2.0 sind:

- **AJAX** (Asynchronous JavaScript and XML) - ermöglicht das asynchrone Laden von Inhalten ohne Seitenreload
- **RSS** (Really Simple Syndication) - erlaubt das Abonnieren von Inhalten
- **Wikis** - kollaborative Plattformen zur Erstellung von Inhalten
- **Blogs** - einfache Veröffentlichungsplattformen für jedermann
- **Social Networks** - Plattformen für soziale Interaktion und Austausch

## Benutzer als Mitgestalter

Das Web 2.0 zeichnet sich vor allem durch die aktive Beteiligung der Nutzer aus. Anstatt nur Inhalte zu konsumieren, werden sie zu Produzenten. Jeder kann bloggen, in Wikis schreiben, Videos hochladen und Meinungen teilen. Dies hat zu einem fundamentalen Wandel in der Art und Weise geführt, wie wir das Internet nutzen.

## Die Plattform als Paradigma

Anstatt Software als Produkt zu verkaufen, wird im Web 2.0 Software als Dienstleistung angeboten. Unternehmen wie Google, Amazon und Facebook bieten Plattformen an, auf denen Nutzer interagieren und Inhalte erstellen können. Das Netzwerk wird zur Plattform.
